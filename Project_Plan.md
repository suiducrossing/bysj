# 基于 YOLOv11 的智能车牌检测与识别系统 - 毕业设计规划

## 一、 项目背景与目标
本项目旨在研究并实现一个具备较高准确率和实时性的智能车牌自动检测与识别系统。系统以最新的 YOLOv11 目标检测算法为核心，结合 OCR（光学字符识别）技术，实现复杂场景下（如不同光照、角度、遮挡条件）的车牌定位与字符提取。

**核心目标：**
1. 深入理解并应用 YOLOv11 网络结构及其在多尺度特征提取、轻量化设计上的优势。
2. 构建/处理车牌数据集（如 CCPD），完成数据清洗与 YOLO 格式转换。
3. 实现端到端的车牌检测（YOLOv11）与字符识别（OCR）。
4. 对比 YOLOv8、YOLOv10 与 YOLOv11 的性能（Precision, Recall, mAP, FPS）。
5. 开发一个用户友好的可视化交互系统（基于 Streamlit）。
6. 撰写符合本科毕业设计规范的学术论文。

---

## 二、 技术路线选型
*   **编程语言**：Python 3.8+
*   **深度学习框架**：PyTorch
*   **目标检测算法**：YOLOv11（核心），YOLOv8、YOLOv10（对比基线）
*   **字符识别算法 (OCR)**：PaddleOCR（支持中文车牌，开箱即用，识别率高）
*   **数据集**：CCPD (Chinese City Parking Dataset，涵盖丰富场景的中国车牌数据集)
*   **可视化交互界面**：Streamlit（纯 Python 编写，适合快速构建美观的机器学习 Web 应用）

---

## 三、 项目阶段规划

### 阶段一：前期准备与数据处理
*   环境搭建（PyTorch, Ultralytics, PaddleOCR 等）。
*   下载 CCPD 数据集。
*   编写 Python 脚本，解析 CCPD 文件名中的标注信息，并转换为 YOLO 训练所需的 `.txt` 格式。
*   划分训练集、验证集和测试集。

### 阶段二：核心检测模型训练
*   使用处理好的数据集训练 YOLOv11 模型，实现车牌的精准定位。
*   进行模型调优（超参数调整、数据增强），提升复杂场景下的鲁棒性。
*   训练 YOLOv8 和 YOLOv10 模型，保存权重用于后续对比实验。

### 阶段三：OCR 字符识别模块集成
*   根据 YOLOv11 输出的边界框（Bounding Box），利用 OpenCV 裁剪车牌区域。
*   将裁剪后的图像输入 PaddleOCR 模块，提取车牌文本。
*   实现检测与识别的端到端联调（输入原图 -> 输出带框图片 + 车牌号）。

### 阶段四：对比实验与性能分析
*   在测试集上评估 YOLOv8, YOLOv10, YOLOv11 的各项指标：Precision, Recall, mAP@0.5, mAP@0.5:0.95, FPS。
*   分析光照、角度、遮挡等因素对检测和识别精度的影响。
*   整理实验数据，生成对比图表（折线图、柱状图等），为论文撰写提供素材。

### 阶段五：可视化交互系统开发 (Streamlit)
*   **界面设计**：侧边栏控制台 + 主展示区。
*   **核心功能**：
    *   支持本地图片/视频上传。
    *   支持模型切换（YOLOv8/v10/v11），方便答辩时展示对比效果。
    *   支持置信度阈值动态调节。
    *   实时展示原图、检测结果图（画框）以及识别出的车牌号码和推理耗时。

### 阶段六：论文撰写与答辩准备
*   撰写毕业论文（绪论、算法原理、系统设计、实验分析、总结）。
*   规范图表和数据，完成排版与查重。
*   制作答辩 PPT，准备系统演示。
